# mdn-ads-block-brave
Block ads on MDN in Brave Browser
## How to use
<img width="682" height="192" alt="image" src="https://github.com/user-attachments/assets/bd1fc9d0-8607-4483-a4db-b7191dfbe111" />
Go to Settings for [Sheilds Filters](brave://settings/shields/filters).
Look for Add custom filter lists.
In the input box, paste https://raw.githubusercontent.com/jhi2/mdn-ads-block-brave/refs/heads/main/block.txt.
You should now have all ads on MDN blocked. If you find more, tell me by opening an issue and I will fix it.
